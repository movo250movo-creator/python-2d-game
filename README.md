# Block Dodger

A simple 2D dodge game built with Python and Pygame.

Move the blue player around the screen and avoid the falling red blocks. Your score increases whenever an enemy passes off the bottom of the screen, and the game becomes faster every 10 points.

## Features

- Movement using the arrow keys or WASD
- Randomly spawning enemy blocks
- Score and speed display
- Increasing difficulty
- Game-over screen
- Restart support

## Requirements

- Python 3.10 or newer
- Pygame
